===============================
Project Wiki Lite
===============================

Project Wiki with sqlite


Quickstart
----------

Run the following commands to bootstrap your environment ::

    git clone https://github.com/gnhua/pwlite
    cd pwlite
    pip install -r requirements.txt
    python run.py

Open browser and go to http://localhost:31415.
